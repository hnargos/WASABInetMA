# WASABInetMA

Framework to configure, train and evaluate a neural network for CEST MRI data.
It uses PyTorch for the NN and optional TensorBoard for in training visualization.

The documentation on usage is found in the /doc folder both in doc.md as 
well as in net_config_explanation.yaml. Additionally, the text.ipynb can provide 
some insight on how to utilize parts of the Framework.

This version is a copy specifically for a master's thesis.
